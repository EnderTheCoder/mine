# 命令介绍

#### 	miner

​		权限：除help外，默认op拥有

​		用法

```
/miner add <矿场名称> <起始位置> <结束位置> <传送点> 添加一个矿场， 每个位置是一组坐标
/miner add <矿场名称> 添加一个矿场 依次点击起始位置、结束位置、传送点
/miner remove <矿场名称> 删除一个矿场
/miner edit <矿场名称> <起始位置> <结束位置> <传送点> 编辑一个矿场， 每个位置是一组坐标
/miner tp <矿场名称>  传送到指定名称的矿场
/miner list 列出已存在的矿场
/miner reload 重载插件
/miner help 展示本条信息
```
